# Simulador-lei-de-coulomb
Pequeno programa em Python que permite calcular:

- Força elétrica
- Distância entre cargas
- Valor de uma das cargas

Baseado na Lei de Coulomb:

F = k * (q1 * q2) / r²

## O que pratiquei nesse projeto

- Funções em Python
- Estrutura de menu
- Laço while
- Condicionais
- Organização modular
